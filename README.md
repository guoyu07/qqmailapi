# 腾讯企业邮件API接口
<li>功能简介</li>
<p>腾讯企业邮箱开放协议，包括面向第三方合作应用和面向企业邮用户两类。其中，面向
   企业邮用户的开放协议，将提供给企业邮用户丰富的应用接口，用户可以根据这些接口定制
   自己统一的企业解决方案。<br >
   通过协议接口，企业用户可以实现：<br >
   1) 单点登录<br >
   可以从公司OA 系统、网站一键进入企业邮箱，免去登录过程。<br >
   2) 新邮件提醒<br >
   新邮件将即时在OA 等办公系统提醒你。<br >
   3) 数据同步<br >
   数据同步可以帮助你同步部门成员信息，你还可以创建、删除、修改帐号，同步部门信息等。</p>
